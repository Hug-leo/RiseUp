    **Step 3: Compress all photos to ≤ 150KB**
    Use macOS built-in `sips` to compress:
    ```bash
    for f in "$PDIR"/*.jpg; do
      SIZE=$(stat -f%z "$f")
      if [ "$SIZE" -gt 153600 ]; then
        sips -Z 1200 --setProperty format jpeg --setProperty formatOptions 70 "$f" --out "$f"
        echo "Compressed: $(basename $f) → $(stat -f%z $f) bytes"
      else
        echo "OK (small enough): $(basename $f) — $SIZE bytes"
      fi
    done
    ```

    If `sips` is not available (non-macOS), use ImageMagick:
    ```bash
    for f in "$PDIR"/*.jpg; do
      convert "$f" -quality 70 -resize "1200x>" "$f"
    done
    ```

    ```bash
    done
    ```
